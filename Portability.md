# Portability

Please note that the projects on my profile were, for the most part, written for my own personal use, and I have decided to share their code bases primarily as a reference for someone who wishes to achieve a similar goal.

My code is always intended to be portable in the sense that I avoid use of platform- or architecture-specific features in favor of standardized options, and I try to note when I *do* use a nonportable feature to enhance performance.

These projects will probably work just fine on a number of systems, provided that they provide the usual standard interfaces.

# Testing

I do not have the time or resources to perform extensive testing on different platforms, so the only testing I typically do is performed on my own personal instance of the target (generally the latest version of Ubuntu running on `x86_64` hardware).

# As Always
If you experience issues with portability or weird bugs, please feel free to create an issue and I will try to help you reach a solution.
